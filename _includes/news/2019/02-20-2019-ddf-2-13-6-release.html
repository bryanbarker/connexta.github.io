Distributed Data Framework (DDF) Release 2.13.6 now available.

Date: 20 February 2019

Connexta is pleased to announce, the latest version of DDF is available for download on Github. DDF is an open source, extensible data integration and content management framework. Release date: 20 February 2019, Release Versoin: 2.13.6.

The 2.13.6 release includes Catalog Search improvements and security patches including security improvements in Solr communication. DDF 2.13.6 release notes provide detail. Download from DDF 2.13.6 release.
